## Installation

```
$ npm install fsm.svg
```

## Load

### HTML

```html
<script src="https://cdn.jsdelivr.net/npm/fsm.ai/dist/fsm-ai.common.min.js"></script>
```

### ES6

```js
import FsmAI from "fsm.svg";
```

## Usage

> in your file

```javascript
var fsm = FsmAI.init("#flow");
```
